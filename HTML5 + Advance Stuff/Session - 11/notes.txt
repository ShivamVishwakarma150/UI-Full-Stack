WebSocket API 
-------------

For implement chatting Application. 
1) first we have to prepare chat server using WebSocket API.
        To create chat server we need to use backend technology.
        we can use : Java, DotNet, NodeJS
Here we are using NodeJS.
to implement chatting application we need websocket library.



Web Socket API 
**************
        - Web socket API provided by HTML 5.0
        - Web Socket API, used to build the chat Applications.
        
        Steps :-
        --------

        -> Create the HTTP server 
        -> Create the chat server 
        -> assign the port number to chat server with http server 
        -> request for connection with chat server 
        -> Read the client data 
        -> send the data to another client 

        To implement chat application we need to download one module

        > yarn add websocket 
                (or) 
        > npm install websocket
