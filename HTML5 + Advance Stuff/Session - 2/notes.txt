Structure of webpage
--------------------

DOCTYPE 

ROOT ELEMENT 
    
    HEAD SECTION

    BODY SECTION

DOCTYPE Representing the Version of HTML.
Ex.
    <!DOCTYPE html> ------ HTML5 version 


<html> </html> tag, Representing Root Element

<head></head> tag, used to represent the header section.

<body></body> tag, used to represent the body section.

Ex.
***

<!DOCTYPE html>
<html>
    <head>
        // META DATA
        // Data about webpage  
        // Author of webpage 
        // description of webpage

        <title></title>
        <style></style>
        <link></link>
        <script></script>
    </head>
    <body>
        // main content
        // Ex. images, tables, audio, video,....

    </body>
</html>
